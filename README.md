# Nonlinear-DiD-Cocycles

Raw code scripts applying [cocycles (Dance and Bloem-Reddy, 2025)](https://arxiv.org/abs/2405.13844) to the [nonlinear DiD problem in Torous et al. (2024)](https://arxiv.org/abs/2108.05858) in a continuous treatments setting. Documented Python scripts and slurm scripts may be added later.
